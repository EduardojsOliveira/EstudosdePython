## 📖 Sobre o Repositório
Este espaço foi criado para documentar toda a minha **evolução na linguagem Python**. Aqui, deposito meus estudos diários, resoluções de desafios e projetos práticos. 

O objetivo é transformar este repositório em um histórico real do meu crescimento como desenvolvedor, partindo do absoluto zero até o domínio de ferramentas avançadas.

---

## 🚀 Minha Trilha de Aprendizado
Neste repositório, você encontrará códigos sobre:

* **Fundamentos:** Sintaxe, variáveis e tipos de dados.
* **Lógica:** Estruturas condicionais (`if/else`) e de repetição (`for/while`).
* **Coleções:** Listas, dicionários, tuplas e sets.
* **Modularização:** Criação de funções e uso de bibliotecas.
* **Projetos:** Pequenos sistemas e automações para consolidar o conhecimento.
